Emu64 build under Linux

Required Packages

- qt-sdk
- git
- cmake
- libsdl2-dev
- libsdl2-image-dev
- libquazip-dev
- libpng-dev
- libavcodec-dev
- libavutil-dev
- libswresample-dev
- libavformat-dev
- libswscale-dev

- git clone https://github.com/ThKattanek/emu64.git
- cd emu64
- mkdir build
- cd build
- cmake CMAKE_BUILD_TYPE=Release ..
